# My Streamlit App

## Overview
A Streamlit web application built with Python. Streamlit is used to create an interactive data-driven web app.

## Project Structure
- `main.py` - Main application entry point
- `.streamlit/config.toml` - Streamlit server configuration (runs on port 5000)
- `pyproject.toml` - Python project dependencies

## Running
The app runs via `streamlit run main.py` on port 5000.
